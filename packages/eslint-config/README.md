# @pikacss/eslint-config

ESLint plugin for PikaCSS that enforces static analysis rules for `pika()` calls.

## Installation

```bash
pnpm add -D @pikacss/eslint-config
```

## Usage

```js
// eslint.config.mjs
import pikacss from '@pikacss/eslint-config'

export default [
  pikacss(),
]
```
