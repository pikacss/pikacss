# @pikacss/nuxt-pikacss

Nuxt module for PikaCSS. Automatically registers the Vite plugin and imports generated CSS.

## Installation

```bash
pnpm add -D @pikacss/nuxt-pikacss
```

## Usage

```ts
// nuxt.config.ts
export default defineNuxtConfig({
  modules: ['@pikacss/nuxt-pikacss'],
})
```
